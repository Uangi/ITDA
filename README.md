💻 **참여인원**

| 이름 | 파트 | 역할 |
|----- |	:----------:|----------|
| 박진영 | BE, FE | 특정도시 기온 api로 조회, 데이트 룩 추천, 회원가입, 파일 업로드 |
| 박지민 | BE, FE | 고객센터 게시판, 메인 css|
| 박사온 | BE, FE | 이성 추천 리스트, 매칭 시스템, 커뮤니티 게시판, 신고 기능 |
| 최원재 | BE, FE | 지도 api 활용, 결제시스템, 로그인 |
